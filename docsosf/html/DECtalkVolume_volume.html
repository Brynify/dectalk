Volume [:volume]

The Volume command is a synchronous command that changes the volume settings. DECtalk Software changes the audio system gain in increments from 0 to 99, in decibels. Increments or decrements of 10 to 20 provide a perceptible increase or decrease in volume. The Volume Set command is an absolute command; Volume Up and Volume Down commands are relative commands and increase or decrease the original value. This command does not affect the volume when the application writes wave files or uses the speech-to-memory capability, because scaling is not done to the speech samples based on the Volume command settings.
