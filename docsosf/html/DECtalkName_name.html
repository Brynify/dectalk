Name [:name]

The Name command allows the current speaking voice to be changed to one of ten built-in DECtalk Software voices. The XX represents the speaker name or letter variable for each voice. The letter variable is the first letter of the speaker name. This command is synchronous.

SYNTAX:
[:name XX]
ABBREVIATION:
none
ALTERNATE COMMAND:
[:nXX]
OPTIONS:
Speaker
Variable
Description

PAUL
p
Default male voice

HARRY
h
Full male voice

FRANK
f
Aged male voice

DENNIS
d
Male voice

BETTY
b
Full female voice

URSULA
u
Aged female voice

WENDY
w
Whispering female voice

RITA
r
Female voice

KIT
k
Child's voice

VAL
v
Val’s voice
PARAMETERS:
none
DEFAULT:
PAUL’s voice


EXAMPLES:
[:name KIT] or [:nk]
Note
A user can change any of the voice characteristics of the current speaker by using the Design Voice command. These changes are active only while the current speaker remains current. To save the voice changes, use the save option of the Design Voice command, which saves the changes as the voice of Val.

Click here to see help on Customizing a DECtalk Software Voice for information on the individual characteristics of a speaker or details on how to change a voice.
