Using the DECtalk Software Speak Applet

The Speak applet demonstrates the DECtalk Software voice set. Each picture * displayed in the Speak dialog represents one of the nine built-in voices. You can select a specific voice by clicking on the picture. DECtalk Software then uses the selected voice to speak the contents of a text file displayed in the edit window or from a file you drag and drop from the Windows File Manager into the edit window.

* Note that the faces are cartoon characters when using VGA 16-color mode.

Using the DECtalk Software Sample applet, you can:

Edit and play text

Change the speaking rate

Load a user dictionary

Save text as a WAVE file

Highlight text as it is spoken
