Design Voice [:dv]

The Design Voice command customizes a DECtalk Software voice by selecting and setting speaker-definition options. This command is asynchronous. DECtalk Software voices provide an adequate selection for most applications. However, if you have a special application requiring a monotone or unusual voice, you can use the Design Voice command to modify any DECtalk Software voice. The speaker-definition options and parameters can be entered as a string or one at a time.
