Using the Program Applets to Learn DECtalk Software Basics

The Programming Applet Help introduces the basic operations involved in DECtalk Software voice and program control. If you are a general user, it gives you the information you need to use the Sample, Speak, and User Dictionary Build Tool program applets. If you are an application programmer, and not familiar with DECtalk Software, it introduces the basic program and voice-control functions used to control DECtalk Software from within an application program.

Topics include:

Speaking a file using the Sample applet

Inserting DECtalk Software in-line commands (Syntax Rules)

Changing voice selections

Changing the speaking rate

Overview of the User Dictionary Build Tool

Building a User Dictionary

Using the DECtalk Software Speak applet

Running DECtalk Software from other applications

Using the SAY command-line applet
