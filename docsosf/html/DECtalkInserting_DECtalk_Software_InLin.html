Inserting DECtalk Software In-Line Commands (Syntax Rules)

Before you can use DECtalk Software in-line commands to modify speech output, you need to know the proper syntax with which to include those commands into a text file. The following illustration and table illustrate the rules of DECtalk Software command syntax.
