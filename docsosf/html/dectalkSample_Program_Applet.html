Sample Program Applet

The sample program applet and components are provided to give you some ideas on how to get started with your application. Both the graphic user interface and source components used to develop that user interface are included as part of the sample program. A visual overview of the DECtalk Sample Application dialog is shown below.

See the Using the Program Applets to Learn DECtalk Software Basics for additional description of the sample program. Note that available languages not installed with the DECtalk Software are grayed out on the Languages menu. Languages shown in dark lettering are installed and ready to be used.
