Starting a Language

To start a DECtalk language, use the TextToSpeechStartLang call to pass the call the two-letter name of the language you want to load. After the language is loaded, you receive a handle to that language. If the language is already loaded, you get the previously loaded handle to that language as the return value. The same handle is used for all instances of a language per run. On failure, you receive a handle with the TTS_LANG_ERROR bit set as defined in TTSFEAT.H.

Example:

TextToSpeechStartLang(“uk”)
