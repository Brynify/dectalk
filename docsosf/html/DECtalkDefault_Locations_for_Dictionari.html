Default Locations for Dictionaries

The location of pronunciation dictionaries is determined by the following:

If your text-to-speech engine is a dynamic engine, which uses DLLs, then the default location for the main and user dictionaries is defined in registry entries .

If your text-to-speech engine is a static engine, which does not use DLLs, then the home directory of the statically linked application is checked first, then the OS defined PATH is checked, and then the registry entry is checked.

If your text-to-speech engine is on UNIX, the location is given in dtuser.lib.
