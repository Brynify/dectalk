Rate Selection [:rate]

The Rate command sets the speaking rate in DECtalk. The rate can range from 75 to 600 words per minute. All values outside the range of 75 to 600 will default to the nearest legal value. For example, if you select a speaking rate of [:rate 880] or 880 words per minute, DECtalk Software defaults to 600 words per minute. This command is asynchronous.

SYNTAX:
[:rate DD]
ABBREVIATION:
none
ALTERNATE COMMAND:
none
OPTIONS:
none
PARAMETERS:
Rate in words per minute
DEFAULT:
180 words per minute


EXAMPLES:
[:rate 400]
