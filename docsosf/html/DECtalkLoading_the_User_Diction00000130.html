Loading the User Dictionary

The startup functions attempt to load the DECtalk Software user pronunciation dictionary from the user’s home directory.

$HOME/user.dic

If the dictionary file is found but cannot be loaded, then the startup function returns an error.

Once the startup function has completed successfully, the user can load and unload user dictionaries using the TextToSpeechLoadUserDictionary and TextToSpeechUnloadUserDictionarycalls.
