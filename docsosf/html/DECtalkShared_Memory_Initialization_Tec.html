Shared Memory Initialization Technique (SMIT) and Access32

The Shared Memory Initialization Technique (SMIT) is a method whereby the ability to run a DECtalk Synthesizer on a given machine is controlled using code that the licensee provides and installs on their customer’s systems. SMIT is used for the “Flexible” Access32 licenses (not for standard DECtalk Software, and not for statically-linked licensees).
