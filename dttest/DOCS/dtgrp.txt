Description of group file (*.grp)
=================================

The group file consists a list of case files. 
Each case file will be called to run.

Note: Using leading ";" for comment line.

Example:	

Simple.grp
		dttest "init.tst"     
		dttest "battest1.tst"
		dttest "battest2.tst"
	       ;dttest "battest3.tst"
